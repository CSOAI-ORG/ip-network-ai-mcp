# ip-network-ai-mcp

MCP server for networking tools.

## Tools

- **parse_ip** — Parse and analyze IPv4/IPv6 addresses
- **subnet_calculator** — Calculate subnet details from CIDR
- **cidr_to_range** — Convert CIDR to IP range
- **dns_lookup_data** — DNS hostname resolution

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
